# 今日の一問 📘🎮

不登校・学年に追いつけていない子のための、**まいにち1問だけ**の学習ツール。
クイズを解くように楽しく、ゲーム（ゼルダ・ホグワーツ等）の話題から本物の学びへつなげます。

**👉 スマホ・PCのブラウザで開くだけ。インストール不要・ネット保存不要（進み具合は端末に自動保存）。**

---

## 特長

- 🎲 **科目ミックス** … ことば・理科・世界・社会・生活・ゲーム・算数を自動で混ぜて出題（同じ科目が連続しない）
- 🎮 **ゲームで学ぶ** … パラセール＝重力、魔法薬学＝化学、ダメージ%計算…ゲームから各教科へ
- 🧩 **自動の図解** … ドット・数直線・円グラフ・国旗・時計などを毎問その場で生成
- 🔁 **その場で問題生成＋苦手まで遡行** … 算数は間違えると前提（やさしい所）へ自動で戻る
- 🎚 **レベル選択** … 🌱小学 / 🔥中学 / ⚡高校 / 🎲ぜんぶ
- 🔊 **音声読み上げ** … 漢字が苦手でも進める
- 👪 **保護者レポート** … つまずきポイントと声がけのヒント
- 🏅 やり込み要素 … XP・連続日数・コンボ・バッジ（間違えても減点なし）

## 使い方

- **オンライン版（GitHub Pages）**: 下のURLをスマホで開き、ブラウザの「ホーム画面に追加」でアプリのように使えます。
- **オフライン1ファイル版**: [`standalone.html`](standalone.html) をスマホに送って開くだけ（全部入りの単一ファイル）。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | 本体（UI＋適応エンジン） |
| `skills.js` | 算数スキルの問題ジェネレーター・図解生成 |
| `topics.js` | ゲーム軸＋小学〜高校範囲のトピック（ここに足せば問題が増える） |
| `standalone.html` | 上記すべてを1つにまとめたオフライン用 |

## ライセンス

個人・家庭学習での自由利用OK。
